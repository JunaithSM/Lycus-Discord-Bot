Overview

Lycus Bot is a powerful deadline scheduling and tracking utility for Discord. It registers countdown trackers, announces daily alert posts, and updates them dynamically to keep your server members informed without channel spam.

Slash Commands

Use these slash commands inside your Discord server to manage deadlines dynamically:

Command Description
/deadline add Create and schedule a new deadline tracker.
/deadline list Display all active deadlines and their countdown values.
/deadline edit Modify settings (channel, title, target date) of an existing tracker.
/deadline delete Remove and cancel a scheduled countdown tracker.
/deadline help Open the quick reference help guide inside Discord.

Message Placeholders

You can customize the daily notification messages sent to your Discord channel. Insert these placeholders in the Custom Alert Message field to inject dynamic deadline properties:

Placeholder Discord Rendering Example Output
{title} Plain Text Project Submission
{countdown} Human-readable countdown duration 2 days, 5 hours, 30 minutes
{relative_timestamp} Discord relative timestamp pill in 2 days (updates in real-time)
{full_timestamp} Discord absolute date and time pill Thursday, June 25, 2026 6:30 PM

Clean Channel Editing

To prevent spamming channels with repeated alert pings, Lycus Bot uses an edit-in-place system. When a daily alert is triggered:

  • It checks if a message was already sent for that deadline.
  • If a message exists, it **edits the same message** to show the updated countdown.
  • It only sends a new message if the previous alert message was deleted or doesn't exist.
